Sleep Apnea Detection Project

This project aims to detect sleep apnea using machine learning techniques and physiological data. Sleep apnea is a serious sleep disorder where breathing repeatedly stops and starts during sleep. The goal of the project is to use a dataset of sleep-related information (such as heart rate, oxygen levels, and breathing patterns) to predict whether a person has sleep apnea.

Key Features:

  • Data preprocessing using Python and Pandas to clean and handle missing data.
  • Feature extraction using statistical and time-series analysis techniques.
  • Model building using machine learning algorithms such as Random Forest, Support Vector Machine (SVM), and Regression.
  • Evaluation of model performance using metrics such as Accuracy, Precision, Recall, and F1 Score.
